Simple python script that reads the metadata of photos and videos, and organises them into folders by years

Uses TinyTag, PIL, PIL.ExifTags 

Supported formats:
	
&nbsp; MP3 / MP2 / MP1 (ID3 v1, v1.1, v2.2, v2.3+)
&nbsp; M4A (AAC / ALAC)
&nbsp; WAVE / WAV
&nbsp; OGG (FLAC / Opus / Speex / Vorbis)
&nbsp; FLAC
&nbsp; WMA
&nbsp; AIFF / AIFF-C
&nbsp; JPG
&nbsp; JPEG
&nbsp; PNG
&nbsp; GIF
